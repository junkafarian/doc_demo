Inline API definitions
----------------------

You can write API documentation inline

.. py:function:: run(times[, debug=False])

   Runs the command the number of ``times`` specified. If ``debug`` is set to
   ``True`` pdb will be lauched on error.


.. note:: Inline API definitions have the downside of falling out of date with
          code changes
